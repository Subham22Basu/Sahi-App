# Sahi: Financial Infrastructure for the Gig Economy

### [🚀 View Live Case Study] (inquisitive-melba-71c3ca.netlify.app)

## 📌 Project Overview
**Sahi** is a conceptual fintech product strategy and UX case study designed to solve the financial invisibility of India’s 25 million platform workers. By synthesizing India's Digital Public Infrastructure (DPI)—including UPI, Account Aggregators, and OCEN—Sahi transforms volatile platform labor into verified financial capital.

---

## 🛠️ Key Features & Solutions
* **The Data Haven:** Automated income smoothing that buffers high-earning weeks to provide a stable, "synthetic" salary.
* **The Identity Anchor:** Portable benefits (insurance/pension) tethered to government-backed IDs (Aadhaar/e-Shram) instead of specific gig platforms.
* **The Trust Metric:** A real-time creditworthiness score based on UPI transaction velocity and payout regularity.

---

## 🏗️ Technical Architecture (India Stack)
This project explores the integration of several key Indian digital protocols:
* **UIDAI (Aadhaar):** For frictionless e-KYC and onboarding.
* **Account Aggregator (AA):** Consent-based financial data sharing.
* **OCEN (Open Credit Enablement Network):** To connect workers with low-interest micro-credit.
* **UPI Network:** Analyzing transaction metadata for predictive underwriting.

---

## 📊 Business Strategy
* **Monetization:** LSP commissions via OCEN, B2B SaaS for gig platforms, and premium financial services.
* **Growth:** Future expansion into ONDC for micro-merchants and AI-driven income forecasting.
